Steps to prepare HTML File
1). Use any text editor for wrting html file and save the file with the file extension .html
2). basic synatax of html file is 
                <!DOCTYPE html>
                         <html>
                               <head>
                           
                               </head>
                                       <body>
                                           
                                       </body>
                        </html>
                        
 where > meta information(title, favicon, style, etc.) about the html file is specified in <head> tag
       > all the visible information (image,paragraphs, hypelink, table, etc.) and documents body in specfified in <body> tag
       > <img > tag is used to enter image into html file
       > <a> tag is used to add hyperlink into html file
       > hyperlink can be added to images by using  <a_ _ _ > <img_ _> </a>
       > <p> tag is used to define a paragraph
       > <br/> tag is used for line breaks 
 3). we can customise the html documrent by using different html tag available
 4). we can view the html page by opening the html file by using any browser
 5). we can create other html page similarly and interlink them by using <a> tag
